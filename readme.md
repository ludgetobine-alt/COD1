# Login & Registration Page

Une page web moderne avec un système de Login et Registration sous forme de popup animée.

## Demo

[Voir le site en ligne](https://ludgetobine-alt.github.io/COD1)

## Fonctionnalités

- Popup animée d'ouverture/fermeture
- Formulaire **Login** et **Registration**
- Transition fluide entre les deux formulaires
- Effet **glassmorphism** (flou derrière le popup)
- Labels flottants sur les champs de saisie
- Design **responsive**
- Compatible **Safari / iOS** (`-webkit-backdrop-filter`)

## Structure du projet

```
COD1/
├── index.html       → Structure de la page
├── style.css        → Mise en forme et animations
├── script.js        → Logique d'ouverture/fermeture
├── background.jpg   → Image de fond
├── background.webp  → Image de fond
├── favicon.svg      → Icône de l'onglet navigateur
└── README.md        → Ce fichier
```

## Technologies utilisées

- **HTML5**
- **CSS3** (Flexbox, animations, backdrop-filter)
- **JavaScript** (DOM, addEventListener)
- **Ionicons** (bibliothèque d'icônes)
- **Google Fonts** (Police Poppins)

## Lancer le projet en local

1. Clone le repo :

```bash
git clone https://github.com/ludgetobine-alt/COD1.git
```

2. Ouvre `index.html` dans ton navigateur

> Aucune installation requise !
